JoeCorp

A self-guided exploration into on-premises Windows administration, modeled after a medium-sized business.

Project Documentation

The full JoeCorp deployment is detailed below as a multi-milestone journal, showcasing the step-by-step process of building a modern enterprise environment.

Infrastructure Overview

The virtual environment utilizes Windows Server 2025 and Windows 11 Enterprise clients, representing a modern enterprise setup.

Domain Controllers (DC01 & DC02)

Deployed as redundant, high-availability Domain Controllers using Server Core for reduced attack surface and lower resource consumption.

Server Core Configuration: IP addresses (10.0.0.1 and 10.0.0.2) configured via PowerShell, with primary/secondary DNS servers pointing to peer DC and loopback for replication integrity.

Dedicated GUI Server (SRV01)

Utilizes Desktop Experience (10.0.0.3) to host edge-case applications and provide management GUI. Hosts centralized file share for user profiles and home directories.

Core Accomplishments

  • Active Directory (AD) Deployment: Multi-DC replicated AD forest (joecorp.local)
  • Redundancy and Verification: Confirmed AD replication using repadmin /replsummary
  • User Experience Management: Implemented roaming profiles and home folders on SRV01
  • Security Segmentation: Created distinct OUs with role-based GPOs
